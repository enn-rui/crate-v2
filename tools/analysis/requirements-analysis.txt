# Crate local analysis deps — for running analysis/analyze_all.py (BPM/key/energy + the MuQ-MuLan
# audio-embedding map + sonic clusters + colored waveforms). This is HEAVY (pulls torch + a music
# embedding model); install it in a SEPARATE venv from the app's.
#
# Proven end-to-end on Windows 11 + Python 3.14 (CPU torch), 2026-06. Runs fine with NO GPU:
# MuQ-MuLan embeds at ~2s/track on CPU, so a ~1000-track library indexes one-time in ~30-40 min.
# A GPU just makes it minutes instead.
#
# torch is intentionally NOT pinned here — install the build for your machine FIRST, then this file:
#   * No GPU / macOS / any laptop:   pip install torch --index-url https://download.pytorch.org/whl/cpu
#   * NVIDIA GPU: get the CUDA wheel matching your card from https://pytorch.org/get-started/
#       - recent cards incl. RTX 40/50-series (Blackwell) need CUDA 12.8:
#         pip install torch --index-url https://download.pytorch.org/whl/cu128
# (Embedding auto-uses CUDA when torch.cuda is available; otherwise CPU.)
